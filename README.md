

# WeatherApp

Una aplicacion cuyo objetivo es la búsqueda del clima actual de cualquier país y ciudad del mundo.  
<br/>
Esta desarrollada con JavaScript utilizando React.  
<br />
Los datos son consumidos de la Api "openweathermap"
